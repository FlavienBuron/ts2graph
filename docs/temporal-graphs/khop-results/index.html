K-Hops Experiment Results

This page shows the MAE, MSE, and RMSE values for different k number of neighbors in K-Hops graph techniques, compared with Empty graph (k = 0) and Fully Connected Graph (k = 128). The vertical line indicates the minimal value, and takes the color of either horizontal lines if those are the minimum (and is placed on either end of the axis).
