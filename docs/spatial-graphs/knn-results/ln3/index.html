KNN Experiment Results: 3 GCNConv Layers

This page shows the MAE, MSE, and RMSE values for different k number of neighbors in KNN-based graph techniques, compared with Empty graph (no edges between any nodes) and Fully Connected Graph (all nodes are neighbors). The vertical line indicates the minimal value, and takes the color of either horizontal lines if those are the minimum (and is placed on either end of the axis). NOTE: the x-axis represent the fraction of the possible values for k. The range of possible value in this case is [0, 35] (k=36 would mean allowing self-loops, which are not used here), thus k=1.0 == k=35, k=0.5 == k=18 etc. This is to easily compare techniques that use difference metrics.
