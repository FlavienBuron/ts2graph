KNN Graph Experiments - GRIN

KNN graphs compared against Empty Graph and Fully Connected graph baselines.

This page reports metrics for the imputation of missing values as a function of the KNN parameter (k) over the gaussian affinity. Here k is the percentile over the possible number of neighbors, such that for N nodes real_k=k*(N-1) (or real_k=k*N if self-loops are allowed). Vertical lines indicate the minimum error or maximum correlation; if the minimum/maximum coincides with a baseline, the line adopts its color and is placed at the axis boundary.

The distance matrix is obtained using the Masked Normalized Euclidean Distance, pair-wise comparison between the time series (sensors). Masked refers to the masking of entries that are completely or partially missing. Only full time points can be compared. Normalized refers to the act of normalizing the distance by the number of non-masked entries, such that the distance is less biased toward short sequences

The affinity matrix is obtained using the gaussian kernel (RBF) over the distance matrix. The RBF parameter gamma is set to 0.1
