Location-based Graph Experiments

Threshold-based graphs compared against Empty and Fully Connected baselines.

This page reports metrics for the imputation of missing values as a function of the threshold parameter (k) over the geographical affinity (affinity <= k). Vertical lines indicate the minimum error; if the minimum coincides with a baseline, the line adopts its color and is placed at the axis boundary.

The distance matrix is obtained using the identity function over the geographical distance between the sensors.

The affinity matrix is obtained using the gaussian kernel (RBF) over the distance matrix.
